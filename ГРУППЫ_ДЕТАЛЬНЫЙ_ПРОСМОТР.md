# Детальная система просмотра групп

## Описание
Добавлена детальная система просмотра информации о группах детского сада с полной статистикой, списком детей и воспитателей.

## Новые файлы
- `view/group_detail_view.py` - Детальное представление группы

## Изменённые файлы
- `view/groups_view.py` - Добавлена кнопка "Просмотр" для перехода к детальному представлению
- `settings/groups_settings.py` - Улучшен метод `get_group_by_id` для получения полной информации
- `main.py` - Добавлен атрибут `content_area` для навигации между представлениями

## Функциональность

### Детальное представление группы включает:

#### 1. Основная информация
- Название группы
- Возрастная категория
- Дата создания

#### 2. Вкладка "Дети"
- Список всех детей в группе
- Для каждого ребенка отображается:
  - ФИО
  - Возраст (автоматический расчет)
  - Дата рождения
  - Пол (с иконкой)
  - Кнопка "Подробнее" для просмотра детальной информации

#### 3. Вкладка "Воспитатели"
- Список всех воспитателей группы
- Для каждого воспитателя отображается:
  - ФИО
  - Телефон
  - Email
  - Кнопка "Подробнее" для просмотра детальной информации (образование, опыт работы)

#### 4. Вкладка "Статистика"
- Общая статистика:
  - Всего детей
  - Количество мальчиков
  - Количество девочек
  - Количество воспитателей
- Распределение по возрасту:
  - 0-2 года
  - 3-4 года
  - 5-6 лет
  - 7+ лет

## Использование

### Открытие детального просмотра
1. Перейдите в раздел "Группы"
2. Нажмите на кнопку меню (три точки) рядом с нужной группой
3. Выберите "Просмотр"

### Навигация
- Кнопка "Назад" (стрелка влево) возвращает к списку групп
- Вкладки позволяют переключаться между разделами информации

### Просмотр детальной информации
- Нажмите на кнопку "i" (информация) рядом с ребенком или воспитателем
- Откроется диалоговое окно с подробной информацией

## Технические детали

### Архитектура
- Использован паттерн MVC (Model-View-Controller)
- Представление создается динамически при выборе группы
- Данные загружаются из базы данных через ORM Peewee

### Компоненты UI
- `ft.Card` - для карточек детей и воспитателей
- `ft.Tabs` - для переключения между разделами
- `ft.ListView` - для прокручиваемых списков
- `ft.AlertDialog` - для детальной информации

### Расчет возраста
- Автоматический расчет возраста на основе даты рождения
- Правильное склонение (год/года/лет)
- Поддержка разных форматов дат (YYYY-MM-DD и DD-MM-YYYY)

## Преимущества
- Удобный просмотр всей информации о группе в одном месте
- Наглядная статистика с иконками
- Быстрый доступ к детальной информации о детях и воспитателях
- Интуитивная навигация
